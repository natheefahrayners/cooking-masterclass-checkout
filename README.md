# cooking-masterclass-checkout

## Features
- Dynamic course catalogue displayed in card components.
- Images, titles, instructors and descriptions included.
- Clean, modern layout.
- Fully responsive design.
- Built with Vue 3 + Vite.

## Project Setup
1. Install Dependencies
    npm install
2. Start the development server
    npm run dev
    http://localhost:5173
3. Build for production
    npm run build
